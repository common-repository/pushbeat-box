== Description ==
Push Notifications plugin for your WordPress site.
Available in Chrome (Android and desktop), Firefox (Android and desktop). 

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/pushbeat-box` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->PushBeat Box screen to configure the plugin
1. Input your Application ID.
